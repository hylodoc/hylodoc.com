we're building Xr0 and enjoy blogging about it.

existing SSGs (like Jekyll and Hugo) are extremely powerful but lack features for interacting with readers, such as comments, subscriber emails, metrics and so on. there's nothing wrong with this — they're literally called static‐site generators — but we want more.

on the other hand, although Substack is nice, it isn't built for programmers. there's very poor support for code: no syntax highlighting and the inline code rendering is jarring. we also want to treat our blog posts like text files and manage them with Git.

so we built Substack for programmers 🥳
  • it generates blog sites from Git repos. Markdown files are pages and code looks beautiful.
  • it separates form from content like LaTeX. no need for custom CSS. pick a theme and start writing.
  • some 🔋🔋 included like comments, email lists, metrics, custom domains, RSS, one‐click exporting of all data. every page can be downloaded as a LaTeX‐typeset PDF. plaintext emails are available to make your hardcore-programmer readers respect you.
  • the SSG is open source, but the batteries are for paying subscribers.
how to 🚀 a hylodoc in 5 minutes
  1. make a repo on GitHub with a file called index.md
  2. grant us read access to the repo
  3. subscribe at $10/month to launch. we offer full refunds for 30 days.
